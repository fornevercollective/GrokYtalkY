Install

One clone. One build.

git clone https://github.com/fornevercollective/GrokYtalkY.git
cd GrokYtalkY
go build -o bin/grokytalky .
./bin/grokytalky
./bin/grokytalky serve
./bin/grokytalky watch clip.mp4

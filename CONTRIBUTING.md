# Contributing

1. Fork under **fornevercollective** conventions.
2. `go test ./...` and `go build -o bin/grokytalky .` must pass.
3. Keep companion defaults: compact UI, clamp all lines, no cam auto-start.
4. Brand: **GrokYtalkY** · module `github.com/fornevercollective/grokytalky`.

See [AGENTS.md](AGENTS.md) for agent-oriented rules.
